## Folder Roles

- `config/` - configuration files and app settings.
- `Controllers/` - request or action handlers.
- `Data/` - datasets, persistence helpers, or stored artifacts.
- `main/` - main application logic or entry point files.
- `Messages/` - message templates, prompts, or response text.
- `Tokenziner/` - tokenizer work, experiments, and notebook-based development.
- `training/` - training scripts, experiments, or model-related resources.
